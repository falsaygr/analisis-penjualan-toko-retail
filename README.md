# Analisis Transaksi Retail: Eksplorasi Tren & Perilaku Pelanggan

Proyek ini bertujuan untuk membedah data transaksi ritel guna memahami pola belanja pelanggan. Saya melakukan analisis mulai dari pembersihan data hingga penarikan insight strategis menggunakan **Python**.

## Ringkasan Proyek
Dalam analisis ini, saya fokus pada korelasi antara waktu belanja (bulan), kategori produk yang diminati, serta pengaruh profil gender terhadap pola pembelian.

## Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn
- **Tool:** Jupyter Notebook

## Insight Utama
Berdasarkan observasi data, saya menemukan beberapa pola unik:

1. **Siklus Musiman Clothing:** Terjadi lonjakan volume penjualan yang signifikan pada kategori **Clothing** di rentang bulan **Februari hingga April**.
2. **Anomali Kategori Electronics:** Produk elektronik mencapai puncak penjualan di bulan **Mei, Agustus, November, dan Desember**. 
3. **Keseimbangan Gender:** Meskipun terjadi lonjakan di kategori tertentu, jumlah pembeli antara **Pria dan Wanita tetap seimbang**. Ini menunjukkan bahwa produk unggulan memiliki daya tarik universal.

## Visualisasi
Analisis ini mencakup:
- Tren pendapatan bulanan (Line Chart).
- Perbandingan kategori produk (Bar Chart).
- Analisis korelasi multidimensi menggunakan **Heatmap**.
